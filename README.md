# Expense Tracker (Console Application)

## Опис проєкту
**Expense Tracker** — це консольний застосунок для обліку особистих витрат.

Проєкт дозволяє:
- додавати, редагувати, видаляти витрати
- працювати з категоріями витрат
- аналізувати статистику витрат
- експортувати дані у CSV-файл за обраний період

### Функціональні можливості
- CRUD для категорій витрат
- CRUD для витрат (назва, дата, категорія, опис, сума, валюта)
- Підрахунок загальної суми витрат
- Мінімальна, максимальна та середня витрата по категоріях
- Фільтрація витрат за датою та валютою
- Експорт витрат у CSV-файл

---

## Обрана СУБД
У проєкті використовується **PostgreSQL**.

PostgreSQL обрана тому, що:
- підтримує реляційну модель даних
- забезпечує цілісність даних
- добре працює з Python через бібліотеку `psycopg2`
- дозволяє зручно реалізувати зв’язки між таблицями (витрати, категорії)

## Інструкція запуску

### 1. Клонування репозиторію
```bash
git clone https://github.com/olgagreb/expense-tracker-exam.git
cd expense-tracker-exam
```

### 2. Створення та активація віртуального середовища
```bash
python -m venv .venv
.venv\Scripts\activate
```

### 3. Встановлення залежностей
```bash
pip install -r requirements.txt
```

### 4. Створення бази даних PostgreSQL
```sql
CREATE DATABASE expense_exam;
```

### 5. Налаштування підключення до БД
```python
DB_NAME = "expense_exam"
DB_USER = "postgres"
DB_PASSWORD = "your_password"
DB_HOST = "localhost"
DB_PORT = 5432
```

### 6. Запуск програми
```bash
python main.py
```

## Приклади команд та сценаріїв демонстрації

### Сценарій 1: Робота з категоріями
- Додати категорію (наприклад: "Продукти")
- Переглянути список категорій
- Відредагувати назву категорії
- Видалити категорію

### Сценарій 2: Робота з витратами
- Додати витрату з вказанням:
  - дати
  - категорії
  - суми
  - валюти
  - опису
- Переглянути всі витрати
- Відредагувати витрату
- Видалити витрату

### Сценарій 3: Аналітика
- Перегляд загальної суми витрат
- Перегляд мінімальної, максимальної та середньої витрати по категоріях
- Фільтрація витрат за валютою та датою

### Сценарій 4: Експорт
- Вибір періоду дат
- Експорт витрат у CSV-файл
- Перевірка створеного файлу у папці `export/`
